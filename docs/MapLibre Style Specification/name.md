# `name`

Human-readable label for the style. Purely descriptive — nothing reads it at render time.

- **Type:** string
- **Required:** no
- **Default:** —

```json
{ "name": "OSM Bright" }
```

Used by UIs such as Maputnik and style pickers to identify the style in a list. Keep it short; version tags belong in [`metadata`](metadata.md).
